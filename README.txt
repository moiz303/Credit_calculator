Мой проект - калькулятор по кредитам. Говоря конкретнее, мой проект - программа, считающая кредит, на который Вы покупаете квартиру, которую потом сдаёте в аренду. После вычислений калькулятор выводит табличку с полезными данными - какой еще остался долг по кредиту, какой остался минус и т д. Программа ещё умеет строить график зависимости полной прибыли от времени. При этом, все вводимые параметры можно сохранить в базу данных или выбрать уже существующие. Также можно создать csv-табличку с таблицей в программе, чтобы открыть в более известном Excel-е.
